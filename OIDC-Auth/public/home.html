Your Own OIDC Authorization Server

A self-hosted OpenID Connect identity provider built with Node.js, Express, and PostgreSQL. Issues RS256-signed JWTs, exposes OIDC discovery endpoints, and lets any OIDC-compatible app authenticate users against it.

Create Account OIDC Discovery

RS256 JWT Tokens

Issues signed JWTs using RSA private/public key pairs. Tokens carry standard OIDC claims — sub, email, name, picture, and expiry — and are verifiable by any client using the public JWKS endpoint.

OIDC Standard Endpoints

Implements the OpenID Connect discovery spec — /.well-known/openid-configuration and /.well-known/jwks.json — so any OIDC-compatible client can auto-configure against this server.

Secure Password Storage

Passwords are never stored in plain text. Each password is salted with a random 16-byte value and hashed with SHA-256 before being saved to PostgreSQL via Drizzle ORM.

UserInfo Endpoint

Protected /o/userinfo endpoint accepts a Bearer token, verifies the RS256 signature, and returns the authenticated user's profile data in standard OIDC format.
